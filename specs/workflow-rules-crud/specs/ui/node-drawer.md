# 節點設定抽屜 (Node Drawer)

## 描述
右側滑出抽屜元件，用於配置 React Flow 畫布上被點擊的節點。不遮擋畫布，管理員可同時看到畫布和設定表單。

## 觸發方式
- 管理員在 React Flow 畫布上點擊任一非 Entrance/Exit 節點時開啟

## 關鍵屬性
- 位置：頁面右側滑出
- 寬度：固定寬度（約 400px），不影響畫布可視區域
- 內容：根據節點類型 (ENode) 動態渲染對應的設定表單
- 操作按鈕：儲存節點設定、刪除節點
- 關閉方式：點擊關閉按鈕或點擊畫布空白區域

## 支援的節點類型設定
| 節點類型 | 設定表單內容 |
|---------|-------------|
| EMAIL | 信件主旨、收件人、內容模板 |
| SMS | 收件人號碼、簡訊內容 |
| LINE | LINE 訊息類型與內容 |
| WEBHOOK | URL、HTTP Method、Headers、Body |
| WAIT | 等待時間（數值 + 單位） |
| WAIT_UNTIL | 指定日期時間 |
| TIME_WINDOW | 時間窗口（開始時間 + 結束時間） |
| YES_NO_BRANCH | 條件判斷表達式 |
| SPLIT_BRANCH | 分支比例或條件 |
| TAG_USER | 標籤名稱 |
