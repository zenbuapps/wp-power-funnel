# Clarify Session 2026-04-02 11:00

## Idea

觸發時機點 TriggerPoint 分組 + 搜尋功能。

目前 ETriggerPoint 已擴展至 20 個觸發點（含 5 個枚舉存根），前端 Select 組件以扁平列表方式呈現所有選項。隨著觸發點數量增長，使用者難以快速定位目標選項。

需求：
- 後端 API 回傳觸發點時包含分組資訊（group 欄位）
- 前端 Select 使用 OptGroup 依分類呈現觸發點選項
- Select 加入 showSearch 搜尋功能，支援模糊匹配觸發點名稱
- 所有使用觸發點 Select 的地方（Edit + List）同步更新
- REGISTRATION_CREATED 不出現在選項中
